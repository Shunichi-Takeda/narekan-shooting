# 🥦 ブロッコリーシューター

**by メディア&広報チーム**

ブラウザで遊べるシューティングゲーム。ブロッコリーを飛ばして野菜の敵を撃破しよう！

## 🎮 遊ぶ

**👉 [ブロッコリーシューターをプレイ](https://shunichi-takeda.github.io/narekan-shooting/)**

## 🕹️ 操作方法

| 操作 | キーボード | マウス / タッチ |
|---|---|---|
| 移動 | `←` `→` / `A` `D` | マウス移動 / スワイプ |
| 発射 | `Space` | クリック / タップ |

## 🎯 ゲームの特徴

- **ウェーブ制**: ウェーブが進むほど敵が強く、速くなる
- **コンボシステム**: 連続撃破でスコア倍率アップ（最大10倍）
- **パワーアップ**: 敵を倒すとランダムでドロップ
  - 🔱 トリプルショット
  - ⚡ ラピッドファイア
  - 💪 メガブロッコリー
  - 💚 ライフ回復

## 🏆 ランキング・スコア登録

ゲーム内の **RANKING** ボタンからランキング（TOP 20）を確認できます。

📊 [スコア一覧を GitHub で見る](../../issues?q=is%3Aissue+is%3Aopen+label%3Ascore)

### スコアの登録方法

1. ゲームをプレイしてゲームオーバーになる
2. **「🏆 スコアを登録」** ボタンをクリック
3. **名前（アカウント名）** を入力して「登録する」をクリック
4. GitHub の Issue 作成画面が開くので **「Submit new issue」** をクリック
5. ランキングに反映される 🎉

> **Note**: スコア登録には GitHub アカウントが必要です。ランキングの閲覧は誰でも可能です。

### スコアデータの仕様

| 項目 | 説明 |
|---|---|
| 保存先 | GitHub Issues（`score` ラベル） |
| 表示名 | 登録時に入力した名前（未入力の場合は GitHub ユーザー名） |
| ランキング表示 | TOP 20（スコア降順） |
| データ取得 | GitHub REST API（認証不要、60リクエスト/時） |
| ローカルスコア | ブラウザの localStorage にハイスコアを保存 |
| 名前の保存 | 入力した名前は localStorage に保存され次回以降自動入力 |

### Issue タイトルフォーマット

```
🏆 {スコア} pts | {名前} | Wave {ウェーブ} | Combo x{最大コンボ}
```

例: `🏆 12345 pts | たけだ | Wave 5 | Combo x8`
