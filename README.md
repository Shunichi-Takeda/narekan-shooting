# 🥦 ブロッコリーシューター

ブラウザで遊べるシューティングゲーム。ブロッコリーを飛ばして野菜の敵を撃破しよう！

## 🎮 遊ぶ

**👉 [ブロッコリーシューターをプレイ](https://shunichi-takeda.github.io/narekan-shooting/)**

## 🕹️ 操作方法

| 操作 | キーボード | マウス / タッチ |
|---|---|---|
| 移動 | `←` `→` / `A` `D` | マウス移動 / スワイプ |
| 発射 | `Space` | クリック / タップ |

## 🎯 ゲームの特徴

- **ウェーブ制**: ウェーブが進むほど敵が強く、速くなる
- **コンボシステム**: 連続撃破でスコア倍率アップ（最大10倍）
- **パワーアップ**: 敵を倒すとランダムでドロップ
  - 🔱 トリプルショット
  - ⚡ ラピッドファイア
  - 💪 メガブロッコリー
  - 💚 ライフ回復
